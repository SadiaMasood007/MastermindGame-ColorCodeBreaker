How to Play

  • The computer generates a secret code using 4 colors.
  • You must guess the code in 10 tries.
  • Click a color below and then click a slot to fill it.
  • Black peg = correct color & position.
  • White peg = correct color, wrong position.
  • You win by getting all 4 black pegs!
  • The "Allow Duplicates" toggle changes whether the secret code can contain repeated colors. Set this before starting a new game.
